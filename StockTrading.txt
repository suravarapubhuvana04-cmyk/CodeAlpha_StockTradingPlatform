# Stock Trading Platform

## Overview

The Stock Trading Platform is a Java console application that simulates a basic stock market. Users can view available stocks, buy and sell shares, manage their portfolio, and monitor their investment performance.

## Features

* View market stock prices
* Buy stocks
* Sell stocks
* Track investment portfolio
* Calculate portfolio value
* Display cash balance
* Object-Oriented design
* Console-based interface

## Technologies Used

* Java
* Object-Oriented Programming (OOP)
* HashMap
* Scanner

## Project Structure

```
StockTradingPlatform.java
```

## How to Run

Compile:

```bash
javac StockTradingPlatform.java
```

Run:

```bash
java StockTradingPlatform
```

## Functionalities

* Display available stocks
* Buy stocks
